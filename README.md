# Nom du Projet

**Description :**  
Ce projet est une application web de gestion multi-rôles (Gestionnaire / Technicien) développée avec **Next.js**, **Tailwind CSS**, **Shadcn/UI**, **Clerk** pour l’authentification, et **Prisma** pour la gestion de la base de données.  

L’application permet :  
- Création et gestion d’utilisateurs avec rôles spécifiques.  
- Accès sécurisé aux dashboards selon le rôle.  
- Gestion des données via Prisma et MySQL/PostgreSQL.

## Technologies

- **Framework** : Next.js 14  
- **UI** : Tailwind CSS, Shadcn/UI  
- **Authentification** : Clerk (email/password, social login)  
- **Base de données** : Prisma ORM + MySQL / PostgreSQL  
- **Icones** : Lucide-react  
- **Outils** : TypeScript, ESLint, Prettier
